## 四阶段设计

> **注意**: 实际实现的 Tab 顺序与原设计略有不同：
> - Tab 0: 分镜拆分
> - Tab 1: 分镜图生成
> - Tab 2: 音频生成（使用独立布局）
> - Tab 3: 视频生成

## Store 设计

> **实现说明**: 实际实现使用了数字索引 (0-3) 而非字符串类型来标识当前 Tab，并增加了更多辅助方法。

### workflowSlice

```typescript
interface WorkflowSliceState {
  // 当前 Tab 索引 (0-3)
  // 0: 分镜拆分, 1: 分镜图生成, 2: 音频生成, 3: 视频生成
  currentTab: number;

  // 各 Tab 完成状态
  tabProgress: Record<number, boolean>;
}

interface WorkflowSliceActions {
  // 切换 Tab
  setCurrentTab: (index: number) => void;

  // 标记 Tab 为完成
  markTabComplete: (tabIndex: number) => void;

  // 重置 Tab 完成状态
  resetTabProgress: () => void;

  // 保存状态到 localStorage
  saveWorkflowState: () => void;
}
```

### sidePanelSlice

```typescript
interface SidePanelSliceState {
  // 左侧栏
  leftPanelWidth: number;        // 默认 200px, 最大 400px
  leftPanelCollapsed: boolean;   // 默认 false

  // 右侧栏
  rightPanelWidth: number;       // 默认 200px, 最大 450px
  rightPanelCollapsed: boolean;  // 默认 false
}

interface SidePanelSliceActions {
  // 调整宽度
  setLeftPanelWidth: (width: number) => void;
  setRightPanelWidth: (width: number) => void;

  // 收起/展开
  toggleLeftPanel: () => void;
  toggleRightPanel: () => void;

  // 设置收起状态
  setLeftPanelCollapsed: (collapsed: boolean) => void;
  setRightPanelCollapsed: (collapsed: boolean) => void;

  // 保存状态到 localStorage
  saveSidePanelState: () => void;
}
```

### shotNavigatorSlice

```typescript
interface ShotNavigatorSliceState {
  // 当前分镜
  currentShotId: string | null;
  currentShotIndex: number;      // 1-based

  // 批量选择

  nextShot: (totalShots: number) => void;

  toggleBulkMode: () => void;
  setBulkMode: (mode: boolean) => void;

```

### 其他现有 Slices

实际实现中还包含以下 slices（保持向后兼容）:

- **dataSlice**: 章节数据、资源数据、分镜数据获取
- **generationSlice**: 图片/视频/转场/音频生成逻辑
- **uiSlice**: UI 状态管理（弹窗、编辑器状态等）
- **chapterActionsSlice**: 章节操作（拆分、保存等）

## 响应式设计

### 桌面端 (≥1280px)

- 三栏完整显示
- 底部导航条固定
- 侧边栏可拖动调整

### 平板端 (768px - 1279px)

- 侧边栏默认收起
- 点击图标展开临时面板
- 底部导航条高度减小

### 移动端 (<768px)

- 单栏布局
- 侧边栏改为抽屉式
- 底部导航条改为垂直滚动列表

## 性能优化

1. **虚拟滚动**：底部导航条超过 20 个分镜时启用
2. **图片懒加载**：缩略图进入视口再加载
3. **防抖处理**：侧边栏拖动调整宽度时
4. **任务状态轮询**：ComfyUI 状态 5 秒刷新，可配置
5. **组件懒加载**：非当前 Tab 内容懒加载
