# 布局系统规格说明

> **状态**: 已实现
> **最后更新**: 2026-03-08

## ADDED Requirements

### Requirement: 三栏式主布局

系统应提供三栏式主布局，包含左侧资源栏、中间编辑区和右侧状态栏。

#### Scenario: 基本布局渲染
- **WHEN** 用户进入 ChapterGenerate 页面
- **THEN** 系统应渲染三栏布局：左侧栏 (200-400px)、中间区 (自适应)、右侧栏 (200-450px)
- **NOTE**: 实际默认宽度为 200px，右侧栏最大宽度为 450px

#### Scenario: 侧边栏宽度拖动
- **WHEN** 用户拖动侧边栏边缘的拖动手柄
- **THEN** 侧边栏宽度应实时调整，左侧栏限制在 200-400px，右侧栏限制在 200-450px

#### Scenario: 侧边栏收起/展开
- **WHEN** 用户点击侧边栏的收起/展开按钮
- **THEN** 侧边栏应收起至 48px 宽度或展开至设定宽度，且有平滑过渡动画

#### Scenario: 音频生成 Tab 独立布局
- **WHEN** 用户进入音频生成 Tab (Tab 2)
- **THEN** 系统应使用 AudioGenTab 组件内部的独立三栏布局，而非标准 ThreeColumnLayout

#### Scenario: 中间区域自适应
- **WHEN** 侧边栏宽度调整或收起/展开时
- **THEN** 中间编辑区应自动调整宽度以填充可用空间

---

### Requirement: 响应式布局适配

系统应支持响应式布局，适配不同屏幕尺寸。

#### Scenario: 桌面大屏适配 (≥1920px)
- **WHEN** 屏幕宽度大于等于 1920px
- **THEN** 三栏布局应正常显示，各区域内容完整

#### Scenario: 笔记本屏幕适配 (1280px-1919px)
- **WHEN** 屏幕宽度在 1280px-1919px 之间
- **THEN** 三栏布局应正常显示，侧边栏宽度适当缩小

#### Scenario: 小屏设备适配 (<1280px)
- **WHEN** 屏幕宽度小于 1280px
- **THEN** 侧边栏应自动收起，用户可手动展开查看

---

### Requirement: 布局状态持久化

系统应持久化用户的布局偏好设置。

#### Scenario: 侧边栏宽度持久化
- **WHEN** 用户调整侧边栏宽度后刷新页面
- **THEN** 侧边栏应保持用户上次设置的宽度

#### Scenario: 侧边栏收起状态持久化
- **WHEN** 用户收起侧边栏后刷新页面
- **THEN** 侧边栏应保持收起状态

#### Scenario: localStorage 存储
- **WHEN** 用户首次访问页面
- **THEN** 系统应从 localStorage 读取布局偏好，若无则使用默认值
