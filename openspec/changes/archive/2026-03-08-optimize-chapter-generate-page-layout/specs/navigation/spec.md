# 导航系统规格说明

> **状态**: 已实现
> **最后更新**: 2026-03-08

## ADDED Requirements

### Requirement: 底部固定分镜导航条

系统应提供底部固定分镜导航条，支持缩略图导航和批量选择。

> **注意**: 实际实现中底部导航条支持折叠功能，展开高度为 160px (h-40)，折叠高度为 40px (h-10)。

#### Scenario: 分镜缩略图展示
- **WHEN** 章节包含多个分镜
- **THEN** 底部导航条应横向滚动显示所有分镜的缩略图

#### Scenario: 分镜状态标记
- **WHEN** 分镜处于不同状态 (已完成/当前/待生成/失败)
- **THEN** 缩略图应显示对应的状态标记 (颜色/图标区分)

#### Scenario: 点击切换分镜
- **WHEN** 用户点击某个分镜缩略图
- **THEN** 系统应切换到该分镜，中间编辑区显示该分镜的详细内容

#### Scenario: 当前分镜高亮
- **WHEN** 分镜被切换
- **THEN** 当前分镜的缩略图应有明显高亮样式

#### Scenario: 批量选择模式
- **WHEN** 用户进入批量选择模式
- **THEN** 用户可以点击多个缩略图进行选择，被选中的缩略图显示选中状态

---

### Requirement: 分镜缩略图组件

分镜缩略图应支持多种交互方式。

#### Scenario: 有图时显示图片
- **WHEN** 分镜已生成分镜图
- **THEN** 缩略图应显示该分镜图的缩略版本

#### Scenario: 无图时显示占位符
- **WHEN** 分镜尚未生成分镜图
- **THEN** 缩略图应显示占位符和分镜编号

#### Scenario: 双击大图预览
- **WHEN** 用户双击缩略图
- **THEN** 系统应打开大图预览窗口显示该分镜的完整图片

#### Scenario: 右键上下文菜单
- **WHEN** 用户右键点击缩略图
- **THEN** 系统应显示上下文菜单 (生成/重新生成/删除等操作)

#### Scenario: 虚拟滚动优化
- **WHEN** 分镜数量超过 20 个
- **THEN** 导航条应使用虚拟滚动技术，仅渲染可见区域的缩略图

---

### Requirement: 键盘导航支持

系统应支持键盘快捷键进行分镜导航。

#### Scenario: 左箭头切换上一分镜
- **WHEN** 用户按下左箭头键 (←)
- **THEN** 系统应切换到上一分镜 (如果不是第一个)

#### Scenario: 右箭头切换下一分镜
- **WHEN** 用户按下右箭头键 (→)
- **THEN** 系统应切换到下一分镜 (如果不是最后一个)

#### Scenario: 数字键切换 Tab
- **WHEN** 用户按下 1/2/3/4 键
- **THEN** 系统应切换到对应的 Tab 页 (分镜拆分/分镜图生成/音频生成/视频生成)

#### Scenario: 边界处理
- **WHEN** 用户在第一个分镜按左箭头或在最后一个分镜按右箭头
- **THEN** 系统应保持在当前分镜，不进行切换

---

### Requirement: 四阶段 Tab 导航

系统应提供四阶段 Tab 导航，展示工作流进度。

> **注意**: 实际 Tab 顺序为 分镜拆分(0) → 分镜图生成(1) → 音频生成(2) → 视频生成(3)

#### Scenario: Tab 标签显示
- **WHEN** 页面加载时
- **THEN** Tab 导航应显示四个阶段：分镜拆分、分镜图生成、音频生成、视频生成

#### Scenario: 当前 Tab 高亮
- **WHEN** 用户处于某个 Tab 页
- **THEN** 对应的 Tab 标签应有高亮样式

#### Scenario: 完成状态标记
- **WHEN** 某个阶段的工作已完成
- **THEN** 对应的 Tab 标签应显示完成标记 (如对勾图标)

#### Scenario: Tab 切换动画
- **WHEN** 用户点击不同的 Tab 标签
- **THEN** Tab 页应平滑切换，有过渡动画
